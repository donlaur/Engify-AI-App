# Guardrails by Category

This directory contains processed guardrail content organized by category. Each file contains guardrails in the condensed format (~200-250 words per guardrail) with E-E-A-T signals.

## Format

Each guardrail follows this condensed structure:

- **Slug:** URL-friendly identifier
- **Category & Severity:** Classification
- **Problem:** One-sentence statement
- **Prevention Checklist:** 5-6 actionable items
- **Early Detection:** One line each (CI/CD, Static, Runtime)
- **Mitigation:** 3 steps
- **E-E-A-T Signals:** Experience, Expertise, Authoritativeness, Trustworthiness (one sentence each)
- **Workflows & Pain Points:** Inline links
