## IV. Technical SEO Priorities

### ChatGPT's Quick Wins (Do These NOW)

**1. Internal Linking (Today)**
- Link from existing high-traffic pages (/prompts, /patterns) to new hubs
- Add "Related Guide" boxes to prompt pages
- Target: 10+ internal links per hub page

**2. Schema Markup (Today)**
- BreadcrumbList on all /learn/ pages
- SoftwareApplication on hub pages
- FAQPage on problem sections
- HowTo on getting started guides

**3. Meta Optimization (Today)**
- Title: "[Primary Keyword]: [Benefit] (2025)" (under 60 chars)
- Description: Keyword + benefit + CTA (under 160 chars)
- Example: "Cursor Rules: Stop AI Slop with 30 .cursorrules (2025)"

**4. Page Speed (This Week)**
- Target: <3 seconds load time
- Optimize images (WebP format)
- Use responsive images (srcset)
- Descriptive alt text on all images

**5. URL Structure (Already Good)**
- `/learn/ai-tools/cursor` (hub)
- `/learn/ai-tools/cursor/rules-guide` (spoke)
- Keep descriptive, avoid query strings

### Gemini's Additions

**6. Bi-Directional Linking**
- New → Old: Hub pages link to existing prompts/patterns
- Old → New: Existing pages link back to hubs
- Creates "closed loop" authority flow

**7. Content Freshness**
- Update articles every 3-6 months
- Add "Last updated: [date]" to articles
- Include dateModified in Article schema
