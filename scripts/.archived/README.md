# Archived Scripts

This directory contains one-off, experimental, and migration scripts that are no longer actively used in production.

## Purpose

These scripts were moved here to:
- Reduce linting/TypeScript errors in active codebase
- Maintain clean, professional code structure
- Preserve historical work without cluttering main scripts
- Improve build performance

## Contents

- **analysis/** - One-off analysis scripts
- **bi/** - Business intelligence research generators
- **content-generators/** - Content generation and improvement scripts (one-time use)
- **db-migration/** - Database migration and data population scripts (already executed)
- **development-checks/** - One-off development validation scripts
- **docker/** - Docker build scripts
- **maintenance-oneoff/** - One-time maintenance and validation scripts
- **marketing/** - Marketing launch readiness checks
- **seo/** - SEO validation scripts
- **admin-oneoff/** - One-off admin utilities

## Note

These scripts may have TypeScript errors or outdated dependencies. They are preserved for reference but are not maintained for production use.

If you need to use any of these scripts:
1. Review and update dependencies
2. Fix any TypeScript/linting errors
3. Test thoroughly before use
