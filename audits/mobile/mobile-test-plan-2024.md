- [ ] Search bar full width

- [ ] Long titles wrap correctly
- [ ] Action buttons accessible

- [ ] Pattern cards display properly
- [ ] Code examples scrollable
- [ ] Navigation tabs work
- [ ] No content overflow

### Learn (/learn)
- [ ] RAG chatbot CTA visible

- Text overflow/truncation

- Image scaling
- Modal/dialog sizing
- Navigation menu
